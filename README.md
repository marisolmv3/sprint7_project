# Aplicación Web de Análisis de Vehículos

## Descripción del Proyecto
Esta aplicación web interactiva permite analizar datos de vehículos en venta en Estados Unidos. 

## Funcionalidades
- **Histogramas interactivos**: Visualiza la distribución de diferentes características de los vehículos
- **Gráficos de dispersión**: Explora relaciones entre variables como precio y kilometraje
- **Controles interactivos**: Botones y casillas de verificación para generar visualizaciones dinámicas
- **Selector de columnas**: Permite elegir qué variable analizar

## Tecnologías Utilizadas
- Python
- Streamlit
- Pandas
- Plotly Express

## Cómo usar la aplicación
1. Selecciona el tipo de visualización que deseas ver
2. Usa los controles interactivos para generar gráficos
3. Explora diferentes variables usando el selector desplegable
